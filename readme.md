# mcp.abc

🚧 A tiny experimental MCP (Model Context Protocol) playground built with TypeScript.

---

## 📖 About

**mcp.abc** is a beginner-friendly sandbox project to explore how MCP works using TypeScript.

This repository is intentionally simple and sometimes "dirty" — the goal is to **learn fast**, **break things**, and **understand MCP concepts** through small experiments rather than building production-ready systems.

---

## 🎯 Goals

* Learn MCP architecture and workflow
* Build small experimental tools using MCP
* Test ideas quickly without over-engineering
* Understand how tools, context, and handlers interact

---

## 🧪 What You'll Find Here

* Simple MCP server setup
* Basic tools (toy examples)
* Experimental handlers
* Quick prototypes (may not be clean 😄)

---

## 🛠 Tech Stack

* TypeScript
* Node.js
* MCP SDK (Model Context Protocol)
